# Eco-Tracker
Eco-Friendly Tracker is a simple web application that helps users build sustainable habits by tracking their daily eco-friendly actions. The goal of this project is to encourage small lifestyle changes that positively impact the environment.

💡 Problem Statement
Many people want to live an eco-friendly lifestyle but often forget or lack motivation to follow sustainable habits daily.

✅ Solution
This project provides a simple and interactive platform where users can:

1) Track daily eco-friendly actions

2) Stay consistent with good habits

3) See their progress and improve over time
